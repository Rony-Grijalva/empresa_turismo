# Multiservicios Grijalva - Sistema de Gestión de Reservas

Este es un sistema web integral diseñado para digitalizar y automatizar la gestión de reservas de transporte para la empresa **Multiservicios Grijalva** en Perú.

## Estructura del Proyecto (Monorepo)

- **`/database`**: Archivos de esquema de base de datos SQL y datos de prueba (`schema.sql`, `seed.sql`).
- **`/backend`**: Backend API con Django Ninja.
- **`/frontend`**: Interfaz de usuario SPA en React, desarrollada con Vite y estilizada con Tailwind CSS.

## Requisitos Previos

- [Node.js](https://nodejs.org/) (Versión 18 o superior recomendada)
- [Python](https://www.python.org/) 3.10+
- PostgreSQL
